五、如何发布Wiki
================================================================================

方法1: 一键脚本（推荐）
---------------------------------------
./publish-wiki.sh

方法2: Git命令行
---------------------------------------
git clone https://github.com/qq940500529/Equipment-Fault-Statistics.wiki.git
cd Equipment-Fault-Statistics.wiki
cp ../wiki/*.md .
rm README.md
git add . && git commit -m "发布Wiki" && git push

---------------------------------------
1. 在Settings中启用Wikis
2. 访问Wiki标签
3. 手动创建每个页面
4. 复制对应的.md文件内容
