## 🎯 核心内容概览

### 1. 项目定位

**设备故障统计系统** - 纯前端Web应用

- **目标**: 将Excel VBA数据处理脚本转换为Web应用
- **特点**: 纯前端、零部署、离线可用、数据本地处理
- **用户**: 设备管理人员、维修统计人员

### 2. 技术架构

#### 技术栈
```
前端框架: 原生JavaScript (ES6+) ✅
Excel处理: SheetJS (xlsx) ✅
图表库: Apache ECharts ✅
日期处理: Day.js ✅
UI框架: Bootstrap 5 ✅
```

#### 架构层次（Phase 1 实施状态）
```
┌─────────────────────┐
│   用户界面层 ✅      │ ← HTML框架、CSS样式、响应式设计
```

### 3. 核心功能（Phase 1-9 分阶段实施）

#### Phase 1 已实现 ✅

### 4. 数据处理流程（逐步实施）

```
Excel文件上传 (Phase 2)
    ↓
SheetJS解析 → 原始数据 (Phase 2)
    ↓
列识别 → 列映射对象 (Phase 2)
    ↓
数据验证 → 验证结果 (Phase 3)
    ↓
数据转换 → 处理后数据 (Phase 4)
    ├→ 删除合计行
    ├→ 车间分列
    ├→ 维修人分类
    ├→ 时间计算
    └→ 删除不完整行
    ↓
┌─────┬─────┬─────┐
↓     ↓     ↓     ↓
预览  图表  Excel CSV (Phase 5-7)
```
