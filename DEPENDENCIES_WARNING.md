# 依赖警告说明

## npm install 时的警告

在运行 `npm install` 时，您可能会看到以下警告信息：

### 1. 已废弃的包警告

```
npm warn deprecated inflight@1.0.6
npm warn deprecated glob@7.2.3
npm warn deprecated abab@2.0.6
npm warn deprecated domexception@4.0.0
```

**说明**：这些是 `jest` 测试框架依赖的传递依赖项。这些警告不会影响应用的正常运行，仅影响测试环境。

**影响**：无影响。这些包仅用于开发测试环境，不会包含在生产构建中。

### 2. 安全漏洞警告

```
1 high severity vulnerability in xlsx
```

**说明**：xlsx 库存在已知的安全问题（原型污染和 ReDoS）。

**缓解措施**：
- 本应用仅在客户端浏览器中使用 xlsx 库
- 不涉及服务器端处理
- 数据不上传到任何服务器
- 用户仅处理自己的本地文件

**官方状态**：目前 xlsx 维护者尚未发布修复版本。我们持续关注更新。

**是否需要修复**：
- 对于本地使用的应用：不需要立即修复
- 如果部署为公共服务：建议等待官方修复或寻找替代方案

## 如何查看详细信息

```bash
# 查看完整的审计报告
npm audit

# 查看建议的修复
npm audit fix
```

## 建议

1. **开发者**：可以忽略这些警告继续开发
2. **用户**：应用功能不受影响，可以正常使用
3. **未来**：我们会持续关注依赖更新，及时升级到安全版本
