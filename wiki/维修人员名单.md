## 分类逻辑

系统使用以下逻辑进行分类：

```javascript
function classifyRepairPerson(name) {
  const trimmedName = name.trim();
  
  // 检查是否为维修工
  if (REPAIR_WORKERS.includes(trimmedName)) {
    return '维修工';
  }
  
  // 检查是否为电工
  if (ELECTRICIANS.includes(trimmedName)) {
    return '电工';
  }
  
  // 不在名单中
  return '未知';
}
```

## 注意事项

### 姓名匹配规则

- ✅ **完全匹配**: 姓名必须完全一致（包括字符）
- ✅ **自动去空格**: 系统会自动去除姓名前后的空格
- ❌ **区分大小写**: 虽然是中文，但仍需准确
- ❌ **不支持别名**: 不会识别昵称或简称

## 如何更新人员名单

如果需要添加或修改人员名单：

### 方法1: 修改源代码

编辑 `js/config/constants.js` 文件：

```javascript
export const REPAIR_WORKERS = [
  '王兴森',
  '孙长青',
  '徐阴海',
  // ... 其他现有人员
  '新维修工姓名'  // 添加新人员
];

export const ELECTRICIANS = [
  '李润海',
  '赵艳伟',
  // ... 其他现有人员
  '新电工姓名'  // 添加新人员
];
```

## 未知分类的处理

如果维修人被分类为"未知"：

### 原因
- 姓名不在预定义名单中
- 姓名拼写错误
- 姓名格式不规范

### 影响
- ✅ 数据仍会被正常处理
- ✅ 可以正常导出
- ⚠️ 在统计图表中会单独显示为"未知"类别

### 建议
1. 检查Excel中的姓名拼写
2. 确认是否需要将该人员添加到名单中
3. 如果是临时人员，可以保持"未知"状态
